## 🧠 Project Overview  

This project delivers a **comprehensive Data Analytics solution** designed to transform fragmented supply chain data into actionable business intelligence.

It covers the full data lifecycle from **data engineering → analysis → visualization → decision support**.

---

### 🚀 Multi-Layer Architecture  

- 🗄️ **SQL Layer:** Data Engineering & Star Schema Modeling  
- 🐍 **Python Layer:** Exploratory Data Analysis (EDA) & Insights  
- 📊 **Excel Layer:** Operational KPI Dashboard  
- 📈 **Power BI Layer:** Executive Business Intelligence Dashboard
